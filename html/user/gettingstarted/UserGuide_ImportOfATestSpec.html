User Guideline: Import of a test specification

In a few steps, you import a test specification. You can also use the cheatSheet.

Open menu

First, go with the mouse courser in the GlobalTester Navigator window. Right-click, and choose "Import".

Select TestSpecification

In the tree view, select "GlobalTester > Import TestSpecification from Plugin". Click the "Next" button.

WizardPage Title

In the text field "Projectname", choose a name for your project. You can import the same TestSpecification under different names in your workspace. In the drop-down menu, you can choose the TestSpecification you want to import. A description of it is given in the field below. Then click "Finish".
