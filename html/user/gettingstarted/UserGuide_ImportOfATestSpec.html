Import of a TestSpecification

In a few steps, you will be able to import a TestSpecification. In order to get a quick overview, you can also use the Cheat Sheet.

Open menu

First, go with the mouse cursor in the "GlobalTester Navigator" window. You might already have some TestSpecifications or TestCampaigns created there. Right-click, and choose "Import". The import wizard will open.

Select TestSpecification

In the tree view, select "GlobalTester > Import TestSpecification from Plugin". Click the "Next" button.

Choose a Project Name

In the text field "Projectname", choose a name for your project.
If the name you have chosen is already existing in your GlobalTester 3 runtime workspace, you will be notified.
In the list, you can choose the TestSpecification you want to import. You can import the same TestSpecification under different names in your workspace.
The name of the bundle the TestSpec is contained in is also displayed. A description of the TestSpecification is given in the field below. You can get a quick overview of the purpose of the TestSpecification there.
Then click "Finish".
