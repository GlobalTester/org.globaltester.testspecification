Create and Export a TestSpecification

In a few steps, this user guide will show you how to create and export a TestSpecification. In order to get a quick overview, you can also use the CheatSheet.

Create your own TestSpecification

First, go with the mouse courser in the "GlobalTester Navigator" window. Right-click, and choose "New -> Project...". In the "New Project" wizard, select "GlobalTester -> TestSpecification", and click "Next >". Enter a name for your TestSpecification Project in the text fiel "Project name". Then click "Finish"

Add a TestCase

In the "GlobalTester Navigator" window, your own TestSpecification will be shown as a project folder. In its sub-folder TestCases, you can now add the default TestCase. TODO: Describe TestCase-wizard.
