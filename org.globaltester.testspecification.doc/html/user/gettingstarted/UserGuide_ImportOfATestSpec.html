Import of a TestSpecification

In a few steps, you will be able to import a TestSpecification. In order to get a quick overview, you can also use the Cheat Sheet.

Open menu

First, move the mouse cursor into the "GlobalTester Navigator" window. You might already have some TestSpecifications or TestCampaigns created there. Right-click, and choose "Import". The import wizard will open.

Select TestSpecification

In the tree view, select "GlobalTester > Import TestSpecification from Plugin". Click the "Next" button.

Choose the TestSpecification you want to import

In the list, you can choose the TestSpecification you want to import. Just below the list you will find the bundle symbolic name of the last selected TestSpecification. Also a description of the TestSpecification is given so that you can get a quick overview of its purpose. For the ease of use several TestSpecifications can be selected for import. If a TestSpecification with the same name is already present within your workspace you will be notified and asked to resolve this conflict. In this case you either can check the checkbox at the bottom of the page which will remove/delete all conflicting TestSpecifications from your workspace prior to importing the new ones. If you want to keep your existing TestSpecifications you are encouraged to rename them in your workspace manually prior to continuing with importing new TestSpecifications Then click "Finish".
